# Satellite Operations (Research & Scraping)

## Objective
To provide autonomous, high-quality intelligence by scraping and analyzing global data sources.

## Workflow
1. **Target Identification**: Extract key research topics from user queries.
2. **Data Acquisition**: Use ScrapingDog API to bypass anti-scraping measures.
3. **Intelligence Synthesis**: Use Gemini/OpenAI to summarize raw data.
